How to Use Your Goal Mandala

  • Double-click on any section to expand it into a quarterly planning grid
  • Click on any goal text to edit it
  • Check the boxes to mark goals as completed
  • Click + Add Goal to add new goals to any section
  • Double-click the expanded section again to collapse back to main view
  • Use the Save Progress button to store in browser
  • Use Save to File to export as JSON file
  • Use Load from File to import saved mandala data
  • Use Clear All to empty all content for a fresh start
  • Use Reset to restore default example content
  • Click Export to save as HTML or print your mandala
  • Try different color themes using the Themes button
